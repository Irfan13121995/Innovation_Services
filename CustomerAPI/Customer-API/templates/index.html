Customer API

Description:

This rest API enables the retrievel of all customers, individual customers and the creation and modification of existing customers.

List all customers

Definition

GET /customers

Response

  • 200 OK on success
[
 {
 "customer_id": "0001",
 "name": "Henk Jansen",
 "address": "Wilgenstraat 45",
 "zipcode": "1234AB",
 "City": "Amsterdam",
 "email": "JADS"
 }
 ]

Creating a new customer

Definition

POST /customers

Arguments

  • "customer_id":string a unique ID for the customer (e.g. c0001)
  • "name":string name of the customer
  • "address":string adress of the customer
  • "zipcode":string zipcode
  • "City":string city
  • "email":string e-mail of the customer

Response

  • 201 Created if succesfully created
{
 "message": "Customer created",
 "data": {
 "customer_id": "0002",
 "name": "Piet Jansen",
 "address": "Wilgenstraat 47",
 "zipcode": "1234AB",
 "City": "Amsterdam",
 "email": "test2@domain.com"
 }
}

Retrieve a customer

GET /customer/<customer_id>

Response

  • 404 Not Found if customer does not exist
  • 200 OK if customer exists
{
 "customer_id": "0001",
 "name": "Henk Jansen",
 "address": "Wilgenstraat 45",
 "zipcode": "1234AB",
 "City": "Amsterdam",
 "email": "JADS"
 }
